Passo a passo — atualizar métricas (a cada 15 dias)

  1. No Instagram, vá em Perfil → menu ☰ → Dashboard profissional → Insights → Account.
  2. Troque o período para 7 dias e anote: contas alcançadas, visualizações, interações, contas engajadas.
  3. Troque o período para 30 dias e anote os mesmos números, mais visitas ao perfil e cliques no link.
  4. Abra este arquivo .html num editor de texto (Bloco de Notas, TextEdit ou VS Code).
  5. Procure o bloco const metricas = {'{'} ... {'}'} perto do fim do arquivo e troque os números.
  6. Salve o arquivo e reenvie pro cliente (ou suba onde ele acessa). Pronto.

Dica: peça pra eu fazer isso — eu consigo abrir o Instagram, ler os números e editar o arquivo pra você.

Como editar as demandas

  1. Direto no site: cada card do Quadro tem um menu de status na parte de baixo — é só escolher o novo status ali que fica salvo automaticamente (visível pra qualquer pessoa que abrir esse link depois). Use o botão + Nova demanda pra cadastrar algo novo. A aba Calendário mostra as mesmas demandas organizadas pelo prazo, mês a mês.
  2. Pelo código (alternativa): abra o arquivo .html num editor de texto, procure o bloco let demandas = [ ... ] perto do fim do arquivo, e edite os campos manualmente. Use isso só se quiser reorganizar tudo de uma vez.

As mudanças feitas no site são salvas num armazenamento na nuvem configurado pela Hadassa.
